# 🚀 Meus Primeiros Passos em Python

Este repositório foi criado para documentar minha jornada inicial na programação com Python. Aqui estão os 5 projetos fundamentais que desenvolvi para consolidar conceitos de lógica, estruturas de dados e automação básica.

O objetivo deste portfólio é demonstrar minha evolução, desde os primeiros comandos de saída até a criação de funções e uso de bibliotecas externas.

---

## 📂 Projetos Iniciais

### 1. 🧮 Operadores e Variáveis (`calculadora.py`)
O início de tudo. Um script focado na manipulação de entradas do usuário (`input`), conversão de tipos de dados (`int`/`float`) e operações matemáticas básicas.

### 2. 🚦 Controle de Fluxo (`verificador.py`)
Aplicação prática de estruturas condicionais (`if`, `elif`, `else`). O projeto foca em como o programa pode tomar decisões baseadas em regras de negócio simples.

### 3. 🔄 Laços de Repetição (`tabuada.py`)
Uso de loops (`for` e `while`) para automação de tarefas repetitivas. Aqui aprendi a controlar o fluxo de execução e trabalhar com contadores.

### 4. 🏫 Portal do Aluno (`dicionario.py`)
Sistema de gerenciamento de notas que utiliza **Dicionários** para associar nomes de alunos (Chaves) às suas respectivas notas (Valores).
* **Conceitos Aplicados:** Manipulação de dicionários, busca de chaves e menus interativos com `while True`.

### 5. 🔐 Gerador de Senhas Aleatórias (`gerador_senhas.py`)
Um utilitário que cria senhas seguras e personalizadas.
* **Conceitos Aplicados:** Importação da biblioteca `random` e `string`, uso de funções (`def`) para organização de código e concatenação de strings dentro de loops.

---

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Bibliotecas Nativas:** `random`, `string`
